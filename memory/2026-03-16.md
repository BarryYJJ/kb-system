## 20:30 ACP hang深入排查
- config.patch加上了 `acp.backend: "acpx"`，但ACP spawn后Claude Code卡在 `[client] session/new (running)` 不动
- **根因确认：Python 3.14.3 SIGABRT崩溃**，发生在 `__kmp_register_library_startup`（OpenMP init），由torch/sentence-transformers C扩展触发，crash在OpenClaw gateway进程内（responsiblePid: 35026）
- 崩溃报告：`/Users/yjj/Library/Logs/DiagnosticReports/Python-2026-03-16-202642.ips` 等三个文件
- **Python切换到3.13**：unlink python@3.14，force-link python@3.13，手动 `ln -sf /opt/homebrew/bin/python3.13 /opt/homebrew/bin/python3` → Python 3.13.12
- 注意：`brew link python@3.13` 不会自动创建无版本号 `python3` symlink，必须手动创建
