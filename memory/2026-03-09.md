## 小红书图文笔记 OCR 处理流程（2026-03-09 总结）

### 适用场景
小红书图文笔记（10张图片以内）的完整内容提取和知识库入库

### 处理流程
1. 用浏览器打开小红书链接（需要登录状态）
2. 用 JS 提取页面中所有图片 URL：
   ```javascript
   Array.from(document.querySelectorAll('img')).map(img => img.src).filter(s => s.includes('xhscdn'))
   ```
3. 用 curl 下载图片到 `/tmp/xhs_imgs/`
4. 用 rapidocr 做 OCR：
   ```bash
   source ~/Desktop/rapidocr_venv/bin/activate
   python3 -c "
   from rapidocr import RapidOCR
   ocr = RapidOCR()
   for i in range(1, 11):
       result = ocr(f'/tmp/xhs_imgs/img_{i}.webp')
       if result and result.txts:
           for txt in result.txts:
               print(txt)
   "
   ```
5. 合并 OCR 文字，用本地向量数据库入库：
   ```bash
   cd ~/.openclaw/workspace/knowledge_bases/personal
   KMP_DUPLICATE_LIB_OK=TRUE python3 kb_local.py add "内容"
   ```

### 关键点
- 需要浏览器保持登录状态（不要关标签页）
- 图片 URL 从页面提取，CDN 无防盗链
- OCR 用 rapidocr 包（不是 rapidocr_onnxruntime）
- 向量入库需要设置 KMP_DUPLICATE_LIB_OK=TRUE（macOS OpenMP 冲突）

### 工具位置
- OCR 虚拟环境：~/Desktop/rapidocr_venv
- 向量数据库脚本：~/.openclaw/workspace/knowledge_bases/personal/kb_local.py
- 向量库路径：~/.openclaw/workspace/knowledge_bases/personal/.vector_db/
