# Domain: Cognition & Personal Management

## Overview
Strategies for managing cognitive resources, mental models, and productivity by resisting algorithmic exploitation and information overload. This domain establishes a framework for prioritizing focus over real-time global processing to mitigate decision fatigue and anxiety.

## Core Architectural Principles
- **Algorithm Resistance**: Active rejection of automated systems as the primary arbiters of information importance. (Ref: `cognition/_index.md`)
- **Noise Filtering (90% Rule)**: Recognition that the vast majority of "breaking" content is irrelevant; critical information is identified through natural repetition rather than immediate consumption. (Ref: `cognition/_index.md`)
- **Capacity Constraints**: Acknowledgement that human cognitive capacity is biologically unsuited for real-time global event processing, necessitating conscious filtering to preserve attention. (Ref: `context.md`)

## Structural Framework
- **Cognitive Flow**: Algorithm Push → Relevance Evaluation → Conscious Filtering → Focus Preservation.
- **Scope**:
    - **Included**: Information consumption patterns, decision-making models, and tech-related mental health management. (Ref: `context.md`)
    - **Excluded**: Technical software architecture and external project management. (Ref: `context.md`)
