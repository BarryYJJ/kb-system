# Domain: Cognition

## Overview
Strategies for managing information consumption to protect cognitive resources against algorithmic exploitation and "breaking news" fatigue.

## Key Principles & Rules
- **Algorithm Resistance**: Do not let automated systems define importance. (Ref: `information_consumption.md`)
- **Information Filtering**: 90% of "breaking" or "urgent" content is irrelevant; truly important information naturally repeats. (Ref: `information_consumption.md`)
- **Focus Preservation**: Conscious filtering is required to prevent anxiety, decision fatigue, and decreased focus caused by real-time global event processing. (Ref: `context.md`)

## Structural Framework
- **Flow**: Algorithm push → Relevance evaluation → Conscious filtering → Focus preservation.
- **Constraints**: Human cognitive capacity is not designed for real-time global processing; focus must be prioritized over algorithmic alerts.
