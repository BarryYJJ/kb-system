--- 正文 ---

刚看完 Andrej Karpathy 的播客，他说他病了，得了严重的 AI 精神病。

他说从12月开始就没手写过一行代码了。每天对着 Agent 说话16小时，同时开十几个 Agent 并行跑任务，token 没花完就焦虑。他管这叫AI精神病。

他说 App 该消失了。智能家居不应该有六个 App，设备只需要暴露 API，Agent 来当胶水层。他三个 prompt 就让 Agent 接管了家里的音响、灯光、空调、窗帘、安防，全部统一到一个 WhatsApp 对话里。

他说客户不再是人类，而是代替人类行事的 Agent。行业需要为 Agent 重新配置，这个重构的规模会相当大。

他说软件需求之所以没有更大，是因为软件太贵了。成本下降之后需求反而会爆发。杰文斯悖论。

他说每个研究组织本质上都可以被描述为一组 markdown 文件。角色、流程、协作方式，全都是代码。一旦是代码，就可以被调优。

他说给人写文档的时代该结束了，以后应该给 Agent 写文档。

这些内容都和《互联网已死，Agent永生》里所写的观点，深深地共鸣了。
