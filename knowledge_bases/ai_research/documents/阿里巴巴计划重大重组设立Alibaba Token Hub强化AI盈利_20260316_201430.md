# 阿里巴巴计划重大重组，设立Alibaba Token Hub强化AI盈利

**来源**: Bloomberg

**类型**: text

**入库时间**: 2026-03-16T20:14:30.144578

---

阿里巴巴集团正在设立一个业务单元Alibaba Token Hub（ATH），将其庞大的AI服务和开发工作纳入统一旗下管理，表明其决心从人工智能中获利。

关键信息：
1. 重组架构：
   - 成立Alibaba Token Hub（ATH），由CEO吴泳铭（Eddie Wu）直接领导
   - 整合Qwen模型研究团队、消费者应用部门、钉钉、夸克品牌设备（如智能眼镜）
   - ATH使命：创建token、传递token、应用token（token即AI计费单位）

2. 战略意图：
   - 明确强调AI变现，部门名称直接引用企业向用户收费的计算单位
   - 增强对企业级AI客户的销售能力，推动Qwen采用
   - 加快研究人员到产品开发各团队之间的互动

3. 盈利困境：
   - 中国消费者不愿为软件订阅付费，AI变现难于西方
   - 大多数中国模型开源且免费下载，与OpenAI/Anthropic收入差距巨大
   - Qwen消费者应用落后于字节跳动豆包，尽管春节投入数十亿红包促销

4. 近期计划：
   - 本周可能宣布基于Qwen的企业级Agent AI服务
   - 由钉钉团队开发，计划逐步整合淘宝、支付宝
   - 2026年通过Qwen在淘宝、高德、支付宝深化agentic AI生态，实现从意图到支付的单次对话转化

5. 人事动荡：
   - Qwen架构师林俊阳（Junyang Lin）本月突然辞职，震动开发者社区
   - 阿里试图平息更多离职传闻

6. 投资承诺：
   - CEO去年承诺AI投资超530亿美元
   - 打造全栈AI产品线包括硬件
   - AGI为公司首要目标

7. 财报时间表：周四公布季度业绩

彭博情报观点：
- Qwen可通过压缩搜索、推荐、结账和履约到一个平台，减少漏斗流失，提升转化率和下单频次
- 随着意图数据积累，阿里可推出更有针对性的福利并减少折扣，维持用户参与度
