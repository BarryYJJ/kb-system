# 每日股市简报

## 概述

开盘前（9:00前）生成核心覆盖标的的每日简报，包含：
- 隔夜外盘表现（美股、港股ADR）
- 关键新闻/消息面
- 机构观点（覆盖标的相关的研报）
- 关键价位提示

## 股票池

38只核心覆盖标的，存储在 `~/Desktop/stock_watchlist_data.py`

## 输出格式

```
📈 每日简报 - YYYY-MM-DD

【隔夜外盘】
- 纳斯达克: +X.XX%
- 标普500: +X.XX%
- 道琼斯: +X.XX%
- 港股ADR: 腾讯 +X.X%、阿里 +X.X%

【关键新闻】
1. [新闻标题]
2. [新闻标题]
...

【机构观点】
- [标的]: [机构] 目标价 [xxx]，评级 [xxx]
...

【关键价位】
- 腾讯控股: 支撑 xxx / 阻力 xxx
- 英伟达: 支撑 xxx / 阻力 xxx
...
```

## 数据来源

- 外盘：Yahoo Finance / 腾讯财经
- 新闻：Web搜索 + 财经网站
- 机构观点：Yahoo Finance recommendations
- 技术位：简单MA5/MA10/前高前低

## 触发方式

- 每天早上 Cron 定时触发（09:00）
- 用户手动请求"每日简报"、"早报"

## 注意事项

- 如果是周末或假期，调整为上周五收盘后外盘
- 简报要简洁，控制在500字以内
- 优先关注有重大新闻或价格异动的标的
