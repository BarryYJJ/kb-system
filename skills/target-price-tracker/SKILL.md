# 机构目标价跟踪

## 概述

跟踪38只核心覆盖标的的机构评级和目标价，提供：
- 实时机构评级（Buy/Hold/Sell）
- 目标价 vs 当前价格
- 目标价上涨空间
- 多机构评级汇总

## 数据来源

Yahoo Finance (info + recommendations)

## 股票池

38只核心覆盖标的，存储在 `~/Desktop/stock_watchlist_data.py`

## 输出格式

```
【机构观点】- YYYY-MM-DD
------------------------------------------------------------
标的 | 当前价 | 目标价 | 空间 | 评级 | 机构数
------------------------------------------------------------
腾讯控股 | 506.0 | 753.3 | +48.9% | Strong Buy | 9/38/3/1
阿里巴巴 | 129.9 | 194.9 | +50.0% | Strong Buy | 6/27/1/0
...
```

## 使用方式

### 查询目标价

用户问"目标价"、"机构评级"、"机构观点"时：
1. 用 yfinance 获取每只股票的 info (targetMeanPrice, recommendationKey)
2. 计算目标价空间 = (目标价 - 当前价) / 当前价 * 100%
3. 获取 recommendations 汇总（strongBuy/buy/hold/sell/strongSell）
4. 按目标价空间排序输出

### 设置提醒

用户说"xxx目标价到xxx提醒我"时：
1. 获取当前目标价
2. 设置提醒条件（上涨空间 > X%）
3. 写入 Apple Calendar

### 触发方式

- 用户手动请求"目标价"、"机构评级"
- 每日简报中自动包含
- Cron 每周更新一次

## 注意事项

- yfinance 数据可能有延迟
- 部分港股/A股可能无机构覆盖
- 目标价是多家机构平均值
