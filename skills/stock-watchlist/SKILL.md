# Stock Watchlist - 核心覆盖标的汇报

当用户提到以下内容时，必须触发此 skill：

## 快速开始

运行股票监控脚本：
```bash
python3 ~/Desktop/stock_watchlist.py
```

## 脚本结构（2026-03-02 重构版）

### 单一股票池数据源
脚本使用 `WATCHLIST` 列表定义所有股票，包含：
- ticker: 股票代码
- name: 名称
- sector: 板块
- market: 市场（A股/港股/美股）
- is_new: 是否新股

**以后加股票只需要在 WATCHLIST 列表中加一行**，自动派生 a_codes、板块列表等。

### 数据获取优化
- A股：mootdx 一次获取
- 港股/美股：并发获取（ThreadPoolExecutor），一次请求缓存结果
- 涨跌幅：用 yfinance.info 的 `regularMarketChangePercent`（实时）

## 输出格式

脚本输出包含：
- 时间戳和市场状态（**同时显示A股、港股、美股的开盘状态**）
- **动态数量的股票**（价格 + 涨跌幅）
- 涨幅 Top 5
- 跌幅 Top 5
- 汇总（上涨/下跌数量）
- **板块分析同时显示涨跌**（如：AI/算力: 涨 3 (...) / 跌 7 (...)）

## 汇报格式要求

⚠️ **硬性要求：必须原样输出脚本全部内容，不能自己精简或加工！**

当用户问"核心覆盖标的怎么样"、"股票行情"等，必须：
1. 运行 `python3 ~/Desktop/stock_watchlist.py`
2. **原样输出完整结果**，包含：
   - 标题（✅ YYYY-MM-DD HH:MM 核心覆盖标的汇报）
   - 时间戳和市场状态（**必须包含A股、港股、美股三个市场的状态**）
   - **全部股票**（价格 + 涨跌幅，数量动态）
   - 涨幅 Top 5
   - 跌幅 Top 5
   - 汇总（上涨/下跌数量）
   - 板块分析（**必须同时显示涨和跌**）

**绝对禁止**：

- 只输出精简版

## 市场状态说明

脚本输出的市场状态格式：
```
北京时间 HH:MM | A股 开盘中/已收盘 | 港股 开盘中/已收盘 | 美股 开盘中/已收盘/盘前 (美东 HH:MM)
```

示例：
- `北京时间 14:30 | A股 开盘中 | 港股 开盘中 | 美股 盘前 (美东 01:30)`

## 定时任务配置

建议 Cron 配置（北京时间，周一至周五）：
- 07:30 - 早间汇报
- 09:20 - 盘中更新
- 09:31 - 开盘后更新
- 10:00 - 上午收盘
- 12:00 - 午间
- 13:30 - 下午开盘
- 14:00 - 盘中
- 15:05 - 收盘
- 16:15 - 盘后

**注意**：周六、周日不发送汇报
