### NemoClaw 产品拆解（占文章40%篇幅）

#### 1. OpenShell（核心资产）
- NVIDIA 开源 Agent 安全运行时（Apache 2.0）
- 架构：Gateway（控制面板）+ Sandbox（隔离容器）+ Policy Engine（策略引擎）+ Privacy Router（隐私路由）
- 底层：K3s 轻量 K8s 跑在 Docker 容器内
- 四层防护：文件系统（创建时锁定）+ 网络（热更新，HTTP方法+路径级）+ 进程（Landlock + seccomp + netns）+ 推理（劫持到受控后端）
- YAML 声明式策略，运行时热加载
- 凭证管理：不落盘，环境变量注入，支持自动发现 Claude/Codex/OpenCode 凭证
- GPU 支持：--gpu 透传宿主机 GPU
- 支持 Agent：Claude Code、OpenCode、Codex、OpenClaw
- 安装：curl 一行命令，开源社区版和 BYOC（Bring Your Own Container）
- 内置实时 TUI（类似 k9s）
- Agent-first 开发：自带 agent skills 做自动诊断、策略生成、安全审查
- Alpha 阶段，目前单玩家模式
- GitHub: NVIDIA/OpenShell

#### 2. NemoClaw
- OpenClaw 的安全增强插件
- 组件：Plugin（TypeScript CLI）+ Blueprint（Python 编排）+ Sandbox（OpenShell 容器）+ Inference（NVIDIA Cloud）
- 一键安装，需要 Linux Ubuntu 22.04+、Node.js 20+、Docker、OpenShell
- 只支持 Linux，需要全新 OpenClaw 安装
- Alpha 阶段
- GitHub: NVIDIA/NemoClaw

#### 3. Nemotron 3 Super 模型
- 120B 总参，12.7B 激活（MoE），Mamba + Transformer 混合架构
- 25T+ tokens 训练，原生 NVFP4 量化训练
- 1M 上下文窗口，支持7种语言（含中文）
- Benchmark：PinchBench 85.6%（开源第一），SWE-Bench Verified 60.47%（开源第一），RULER 1M 91.75%（开源第一），DeepResearch Bench #1
- Artificial Analysis Intelligence Index 36分（比上代高12分）
- API 速度 442 tok/s（全平台第二快）
- API 完全免费
- 本地部署：64GB 内存起步（GGUF），Mac MLX 9-bit 量化版约 33.7 tok/s
- 对比 Claude 4.1 Opus：知识任务 52.8 vs 57.6（Opus 领先5分）
- Agent 场景强在"稳定性"而非"智能峰值"
